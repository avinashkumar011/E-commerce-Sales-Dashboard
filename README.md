# E-commerce Sales Dashboard

## 📊 Project Overview
This project analyzes store sales data using Excel.

# 🔧 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Dashboard

# 📈 Insights
- Women customers contribute 64% sales
- Top states: Maharashtra, Karnataka
- Amazon is highest sales channel
